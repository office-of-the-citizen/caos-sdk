01What the SDK guarantees

This is not a convenience wrapper. The SDK is the constitutional boundary between applications and the operating system — it carries truth out, and nothing in.

✓

Truth is projected, never composed

Applications receive finished public records. Ranking, verification and vocabulary arrive on the projection — the SDK adds no opinion of its own.

◈

Permanent geography, offline

The published Engine 10 permanent snapshot ships inside the package: every state and Local Government resolves synchronously, with no network.

λ

Contracts are executable

Every payload is validated by zod schemas exported from the package — the same schemas the operating system emits against.

⛨

Honest silence

UNKNOWN is a governed state. Missing truth arrives as explicit missingness postures, never as nulls an application must guess about.

02Installation

The package is published to GitHub Packages under the @office-of-the-citizen scope.

# .npmrc — point the scope at GitHub Packages
@office-of-the-citizen:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=ghp_your_token_here

# then
npm install @office-of-the-citizen/caos-sdk
Authentication needs a personal access token with the read:packages scope. In CI, prefer NODE_AUTH_TOKEN.

03Quick start

Resolve a place from the permanent snapshot, then overlay live constitutional truth.

import {
  CaosClient,
  getBySlug,
  breadcrumb,
  lgaNavigationTree,
} from "@office-of-the-citizen/caos-sdk";

// 1. Permanent identity — synchronous, offline, always available
const place = getBySlug("lg-oy-ibadan-north");
const lineage = breadcrumb(place.canonical_id);

// 2. Live truth — through the Engine 12 gateway (may be unavailable)
const caos = new CaosClient("https://gateway.example.ng");
const record = await caos.getPublicRecord(place.slug);

// 3. Continue rendering when truth is missing — the record is honest about it
console.log(record.sections.chairman.badge.label); // "Verified" | …
Composition belongs to applications: load the permanent shell first, overlay truth when reachable, and never collapse a page because the gateway is down.

04API reference — public surface

network methods speak to the gateway. local functions resolve from the packaged snapshot with no I/O.

CaosClient

new CaosClient(baseURL, options?)
Construct a client against a gateway root. Options carry a session token or API key for operator surfaces.
getPublicRecord(slug)
One finished public record — sections, context, budget, activity, placeholders, vocabulary, provenance.network
getPublicNavigation()
The national navigation index: states and their Local Governments, as the OS projects them.network
searchPublicRecords(query, options?)
Engine 11 search over prepared public records. Ranking is OS truth — clients only render.network
getRuntimeIdentity()
Gateway version handshake for the compatibility gate.network

Permanent snapshot

getBySlug(slug) / getByCanonicalId(id)
Resolve one permanent spatial object — name, lineage, area, wards, polling units.local
breadcrumb(canonicalId)
Root-first constitutional lineage: Nigeria → State → Local Government.local
lgaNavigationTree()
Every state with its Local Governments — the offline discovery structure.local
listByClassification(kind) / listChildren(id)
Enumerate governed objects by class, or the children of any object.local
getPermanentSnapshotMeta()
Snapshot epoch, generation time, object counts and content hash — verify what you ship.local
The snapshot deliberately contains no population figures. Population is live truth and arrives only on projections, with its own verification posture.

05Contracts

Every shape the OS emits is exported as a zod schema and a TypeScript type. Validate at the boundary; render with confidence. 

import {
  PublicRecordSchema,
  NavigationIndexSchema,
  SearchResponseSchema,
  type PublicRecord,
  type RecordSection,
  type BadgePresentation,
} from "@office-of-the-citizen/caos-sdk";

const parsed = PublicRecordSchema.safeParse(payload);
if (!parsed.success) {
  // the transport lied — render the governed unavailable state
}

06Error model

Every transport failure normalises to a single CaosError — code, message, HTTP status and the raw envelope. No stringly-typed guessing. 

try {
  await caos.getPublicRecord("lg-oy-ibadan-north");
} catch (e) {
  if (e instanceof CaosError) {
    e.code;    // "HTTP_503" | "NETWORK_ERROR" | governed code
    e.status;  // 503
    e.detail;  // operator-facing detail when provided
  }
}

07Versioning & the compatibility gate

The SDK knows which gateway range it speaks (CAOS_REQUIRED_GATEWAY) and exposes its own build as CAOS_SDK_VERSION. Surface both in your footer; refuse silently-wrong conversations. 

import { CAOS_SDK_VERSION, CAOS_REQUIRED_GATEWAY } from "@office-of-the-citizen/caos-sdk";

const identity = await caos.getRuntimeIdentity();
// identity.gatewayVersion is checked against CAOS_REQUIRED_GATEWAY
Projections are derived, disposable and rebuildable. A client that caches a record caches a snapshot of truth — the provenance block tells you exactly which build you hold.
